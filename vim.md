# Vim Cheatsheet

## Basics

Make changes to file, insert mode | i
Exit insert mode | "esc" key
Help | :help
Open File | :e
Save File | :w
Save and close Vim | :wq
Quit and discard changes | :q!
Quit | :qa

## Information

.vimrc is the settings file for Vim
edit .vimrc to make changes to the Vim interface
